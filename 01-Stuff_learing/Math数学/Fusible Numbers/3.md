# $F_2$的性质

1. **良序性 (Well-orderdness)**:

   ${F}_2$ 是一个良序集。这意味着 ${F}_2$ 的任何非空子集都存在最小元。

   > 对于任意燃烧数 $r(w)$ ，所有严格大于 $r(\alpha)$ 的燃烧数构成的集合中也存在一个最小元，这个最小元自然就是 $r(\alpha)$ 的直接后继，记作$s(r(\alpha))=r(\alpha+1)$ 
   >
   > > 定义 $m(r)=s(r)-r$ 

   

   

2. **闭集性 (Closedness)**:

   ${F}_2$ 是实数集上的一个闭集。这意味着 ${F}_2$ 中任意柯西序列（或收敛序列）的极限点也必定属于 ${F}_2$ 。特别的，这保证了如 $r(\omega) = \lim_{n \to \infty} r(n)$ ， $r(\omega^2) = \lim_{n \to \infty} r(\omega \cdot n)$ 等极限燃烧数的存在性（如果这些极限在实数意义下存在）

   

3. **可数性 (Countableness)**:

   ${F}_2$ 是一个可数集。这意味它可以与自然数集合 $\mathbb{N} = \{0, 1, 2, \dots\}$ 建立一一对应关系



# 证明

以下证明可以**推广**至$F_n$

## 良序性证明

>**良序集**有三个等价定义
>
>1. 非空子集有最小元
>2. 不存在无穷递降链
>3. 在其上有超限归纳法成立



将某个燃烧数$r$的生成形式写为不断应用**生成函数$g(x,y)=\frac{x+y+1}{2}$**形成的**(二叉)树**:

- 树的根节点是 $r$ 本身
- 如果 $r = g(x,y)$ ，那么 $x$ 和 $y$ 就是 $r$ 的左右子节点
- 递归地构建 $x$ 和 $y$ 的子树，直到达到叶子节点，叶子节点通常是基础燃烧数 $0$ 



**例如**:
$$
\frac{5}{4} = g\left(\frac{1}{2}, 1\right) =g\left(\frac{3}{4}, \frac{3}{4}\right)
$$
这两种生成方式对应的二叉树如下图所示：

![img](https://pic1.zhimg.com/v2-46458a4a39da38582f5e60edc8600e5e_1440w.jpg)

这类树有以下三个性质：

1. 有根性: $R$ 的每一个节点都能追朔到根节点，也就是 $r$ 

   

2. 有限性: $r$ 是通过有限次迭代生成的，所以树 $R$ 只有有限个节点

   

3. 存在嵌入关系: 如果 $a$ 所对应的图 $A$ 是 $r$ 所对应的图 $R$ 的一部分，或者说 $A$ 可以”嵌入“ $R$ ，则 $r$ 可以经过 $a$ 生成 ，并且有 $a<r$ 

> [!NOTE]
>
> **Kruskal 树定理[[Kruskal Theorem]]**:
>
> 表明对于任何无限序列的有限有根树 $T_1, T_2, T_3, \dots$ ，必定存在 $i < j$ ，使得树 $T_i$ 可以**同胚嵌入**到树 $T_j$ 中
>
> > **同胚嵌入**
> >
> > 比嵌入关系更弱:$R_k$嵌入$R_j$要求$R_j$存在和$R_k$一模一样的子树，而同胚嵌入不必一模一样，而允许再删除一些度为2的顶点0



> **Proof**
>
> 使用反证法:
>
> **假设$F_2$**中存在无穷**递降**序列 $r_1 > r_2 > r_3 > \dots$
>
> 令 $R_n$ 为对应于 $r_n$ 的生成树。这是一个无限的树序列 $R_1, R_2, R_3, \dots$ 
>
> 根据**Kruskal 树定理**，必定存在 $k < j$ ，使得 $R_k$ 可以同胚嵌入到 $R_j$ 中
>
> 根据对燃烧数生成树的观察和生成函数 $g$ 的**单调性**，如果 $R_k$ 可以嵌入到 $R_j$ 中，那么必然有 $r_k \le r_j$ 
>
> 假设的 $r_k > r_j$ 相矛盾
>



## 闭集性证明

> **Proof**
>
> 使用反证法:
>
> **假设**$F_2$不是闭集,则$F^- = \overline{F_2} \setminus F_2$ 非空,设$a_1\in F^-$
>
> 根据 $F_2$ 的良序性，对于任何实数 $x$ ，如果存在大于 $x$ 的燃烧数，那么其中必定存在一个最小的
>
> 对于坏点 $a_1$ ，存在一个最小的燃烧数 $s(a_1)$ 大于 $a_1$ ，因此区间 $(a_1, s(a_1))$ 中并没有燃烧数
>
> 因为 $a_1$ 是 $F_2$ 的聚点，所以必然存在一个由 $F_2$ 中的数组成的序列 $\{r_n\}$ (例如 $r_1, r_2, \dots$ )，并且  $\lim_{n \to \infty} r_n = a_1$ 且$r_n < a_1$ 
>
> 每个 $r_n$ 都是通过 $r_n = g(b_n, c_n)$ 生成的，其中 $b_n, c_n \in F_2$  $b_n < r_n$ 和 $c_n < r_n$ 
>
> 因此 $b_n < a_1$ 且 $c_n < a_1$ 。这意味着序列 $\{b_n\}$ 和 $\{c_n\}$ 都是有界的实数序列
>
> 根据Bolzano-Weierstrass定理，它们各自至少存在一个聚点 $b, c \in \overline{F_2}$ 
>
> > [!NOTE]
> >
> > **Bolzano-Weierstrass定理**: [[Bolzano-Weierstrass Theorem]]
> > 在 $\mathbb{R}^n$ 中，每一个 **有界序列** 都存在一个 **收敛子序列**
>
>  $b$ 和 $c$ 中至少有一个必须是坏点。选择其中一个坏点，记作 $a_2$ 。根据 $g$ 的单调性，可以确保 $a_2 < a_1$ 
>
> > 假设 $b$ 和 $c$ 都不是坏点，则 $b \in F_2$ 并且 $c \in F_2$ 
> >
> > 由于 $g(x,y)$ 函数是连续的，并且存在子序列 $b_{n_k} \to b$ 和 $c_{n_k} \to c$ 使得 $r_{n_k} = g(b_{n_k}, c_{n_k}) \to a_1$ ，那么必然有 $g(b,c) = a_1$ 
> >
> > 如果 $b, c \in F_2$ ，根据 $F_2$ 的定义， $g(b,c)$ 也必须属于 $F_2$ 。所以 $a_1 \in F_2$ 
> >
> > 这与 $a_1$ 是一个坏点（即 $a_1 \notin F_2$ ）的前提相矛盾。
>
> 重复以上过程:找到一个坏点 $a_3 < a_2$ ，以此类推。构造出一个无穷递降的坏点序列： $a_1 > a_2 > a_3 > \dots$ 
>
> 根据实数的完备性，这个无穷递降链必定收敛到一个极限 $a = \lim_{n \to \infty} a_n$
>
> 聚点的聚点 $a$ 也必定是 $F_2$ 的聚点（即 $a \in \overline{F_2}$ ）
>
> 再次运用 $F_2$ 的良序性：存在最小的燃烧数 $s(a)$ 大于 $a$ ，使得区间 $(a, s(a))$ 中没有 $F_2$ 中的点。
>
> 由于 $a_n$ 是一个严格递减且收敛于 $a$ 的序列，对于任何 $\epsilon > 0$ ，总会存在足够大的 $N$ ，使得当 $n > N$ 时， $a < a_n < a + \epsilon$ ,取 $\epsilon = s(a) - a > 0$ 。这样，当 $n$ 足够大时， $a_n$ 就会落入区间 $(a, s(a))$ 中。
>
> 矛盾
>



## 可数性证明

> 显然燃烧数都是有理数,有理数可数,则燃烧数可数



# 求$s(r)$的算法

此算法称作**SUCC(r)**

```text
procedure Succ(r)
  if r < 0 then
    return 0
  min <- infinity  // 初始化找到的最小值为无穷大
  y <- r         // 初始化 y，作为搜索的一个起点/界限
  repeat          // 开始循环搜索
    x <- SUCC(2r - y - 1)   // 递归调用：找到 > 2r-y-1 的最小燃烧数 x
    y <- SUCC(2r - x - 1)   // 递归调用：基于找到的 x，找到 > 2r-x-1 的最小燃烧数 y
    if x ~ y < min then      // 计算这对 (x,y) 产生的结果
      min <- x ~ y           // 如果结果更好（更小但仍 > r），则更新最小值
    y <- y - ruler(x) // 将 y 更新为其在 F 中的前驱 (predecessor)，ruler(x)为直尺函数
  until y <= r - 1/2         // 循环终止条件
  return min                 // 返回找到的最小值
```
