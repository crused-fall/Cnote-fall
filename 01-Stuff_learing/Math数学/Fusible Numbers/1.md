# 燃烧数的起源

由**Carl Morris**提出的“燃绳谜题”（Burning Rope Puzzle）:

假设有两根材质不均匀的绳子，每根绳子从一端点燃恰好可以燃烧一个小时。如何利用这两根绳子精确地测量出45分钟？由于绳子燃烧速率不均匀，我们无法通过观察绳子燃烧的长度来判断时间 。


## 关键信息与约束条件

- **两根绳子：** 你拥有两根可用的绳子。
- **总燃烧时间：** 每根绳子单独从一端点燃，完全烧尽需要 **1小时（60分钟）**。
- **不均匀燃烧：** 绳子的燃烧速度不是恒定的。不能假设烧到一半长度就代表半小时。
- **可多点点燃：** 你可以在绳子的任意一端或两端同时点燃。



## 标准解法思路

**步骤一 (计时开始 - 0分钟):**

- 拿起 **第一根绳子 (绳子A)**，同时点燃它的 **两端**。
- 同时，拿起 **第二根绳子 (绳子B)**，只点燃它的 **一端**。

**步骤二 (计时进行中):**

- 绳子A由于是从两端同时燃烧，其总燃烧时间将是原来的一半，即 60分钟/2=30分钟。

**步骤三 (关键节点 - 30分钟):**

- 当绳子A **恰好** 完全烧尽时，时间正好过去了 **30分钟**。
- 在这一瞬间，立刻点燃 **绳子B的另一端**。

**步骤四 (计时继续):**

- 此时，绳子B已经被燃烧了30分钟（从它的一端开始）。
- 现在绳子B变成了从 **两端同时燃烧**。
- 绳子B剩下的部分原本需要 (60−30)=30分钟 才能烧完（如果只从一端烧的话）。但由于现在是从两端同时烧，所以烧完剩余部分只需要 30分钟/2=15分钟。

**步骤五 (计时结束 - 45分钟):**

- 当绳子B **完全烧尽** 时，从步骤三开始又过去了15分钟。
- 总共测量的时间 = 绳子A烧尽的时间 + 绳子B剩余部分烧尽的时间 = 30分钟+15分钟=45分钟。



# 燃烧数的生成规则

这个谜题启发了伊利诺伊大学的 **Jeff Erickson**将其推广并形式化为一个数学结构



## 基础

**假设我们拥有无限多根完全相同的、燃烧时间为1小时的不均匀绳子**

我们发现可以通过特定的操作来生成可以精确测量的时间长度。我们将这些可测量的时长称为 **燃烧数** **Fusible Numbers**

定义两种基本的生成规则，从初始可测时长 0 开始：



### 规则一：1-燃烧数 ($F_1$)

**这种规则对应于使用一根完整的绳子，如果$a$是一个可测时长，那么在$a$时刻从一端点燃绳子，等待烧完。这样计时精确地增加$1$小时**

- **定义:**
  $0$ 是一个 $1$-燃烧数。
  如果 $a$ 是一个 $1$-燃烧数，那么 $a+1$ 也是一个 $1$-燃烧数

  

- **生成的集合:** 我们将通过此规则生成的燃烧数集合记为$F_1$

  

- **示例:** 从 0 开始，应用规则一，我们可以得到：
  $0\in F_1$

  $0+1=1\in F_1$

  ...

$F_1=0,1,2,3,...=N$ (所有非负整数)。

### 规则二：2-燃烧数 ( $F_2$)

**这种规则对应于使用一根绳子，在两个不同的已知可测时刻****和** **(满足特定条件) 分别点燃两端，得到一个新的可测时刻。**

- **定义:**
  $0$是一个 $2$-燃烧数。
  如果$a$和$b$都是$2$-燃烧数，并且满足条件 ，那么$z=\frac{a+b+1}{2}$也是一个 $2$-燃烧数。我们通常将这个操作记为 $z=a\sim b$。

- **生成的集合:**我们将通过此规则生成的燃烧数集合记为 **F_2。**



- **示例**

  $0\in F_2$

  $0\sim0=\frac{0+0+1}2=\frac12\in F_2\\0\sim\frac12=\frac{0+\frac12+1}2=\frac34\in F_2\\\frac12\sim\frac12=\frac{\frac12+\frac12+1}2=1\in F_2\\0\sim\frac34=\frac{0+\frac34+1}2=\frac78\in F_2\\\frac12\sim1=\frac{\frac12+1+1}2=\frac54\in F_2\\1\sim1=\frac{1+1+1}2=\frac32\in F_2$
  $$
  F_2=\{0,\frac{1}{2},\frac{3}{4},\frac{7}{8},\ldots,1,\frac{9}{8},\ldots,\frac{5}{4},\ldots,\frac{11}{8},\ldots,\frac{3}{2},\ldots\}
  $$
  

## $F_2$的生成公式



**定理**：

集合$F_2\cap(0,1)$中的数恰好是所有形如$1-2^{-k}$的数，其中$k$是正整数($k\in\mathbb{N}^+$ )。即：$F_2\cap(0,1)=\{1-2^{-k}\mid k=1,2,3,\ldots\}=\{1/2,3/4,7/8,15/16,\ldots\}$属于$F_2\cap(0,1)$。



**证明:** 数学归纳法(略)
